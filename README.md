# substrata-server
Test project

> **_Prerequisites_**  
> _1. Node.js installed_  
> _2. npm installed_  


##### Clone repository
Run the follow commands

```
$ git clone https://github.com/mariefalco/substrata-server.git
```


##### Start dev server  
In root directory run the follow commands
```
$ npm i
$ npm run start:dev
```
